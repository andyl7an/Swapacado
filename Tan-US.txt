User Story 1:
As a grocery shopper, when I forget to buy an ingredient, I want to be able to view who’s nearby with the missing ingredient I need for my recipe, so that I can cook my meal.
User Story 2:
As a charitable person, I wanted to be able to give away my unused ingredients to those who need it more, so my ingredients to go to waste.
User Story 3:
As an online shopper, when I browse items, I want to be able to filter them by proximity and value, so I can find what I want.
